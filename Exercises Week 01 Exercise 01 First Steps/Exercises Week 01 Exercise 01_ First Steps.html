Exercises Week 01 Exercise 01: First Steps

Prompt

 

Create your first piece of the web using the online editor CodePen. Using CodePen eliminates the friction of installing and configuring a local setup, providing you with immediate results.

 

Detailed Explanation

 

CodePen is a popular online tool for quickly prototyping and sharing front-end code. For this exercise, you will use its HTML panel to write a few basic elements and see them render instantly. This exercise is designed to give you a low-stakes, hands-on introduction to HTML syntax.

 

Exercise Guidelines

 

Exercise Instructions

 

  1. Navigate to CodePen.io Links to an external site. and create a free account.
  2. Create a new "Pen".
  3. In your new Pen, within the HTML panel, create an <h1> element containing your name.
  4. Below the <h1>, create a <p> element containing one sentence about why you're interested in building websites.
  5. Save your Pen.

 

Note: Refer to the MDN Web Docs for using the <h1>-<h6> Links to an external site. and <p> Links to an external site. elements.

 

Exercise Submission Details

 

Your submitted Pen must contain a valid <h1> element and a valid <p> element with the content specified in the instructions.

  1. Submit the URL to your saved CodePen for this exercise.
